= 修改工作負載以將QoS升級至AQO
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
支援與Unified Manager管理的儲存工作負載相關的傳統QoS和調適性QoS（AQO）Active IQ Unified Manager 。

如果將ONTAP 您的支援叢集從9.1升級至9.3（檔案共用）、將9.4（LUN）、您可以將Unified Manager管理的各個工作負載的傳統QoS升級至AQO。
